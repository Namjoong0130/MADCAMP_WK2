# 👗 MODIF : Modify your mode

> **"같은 남색 가디건인데 왼쪽은 10,000원대, 오른쪽은 160만 원대?"**

고작 줄 4개 긋는다고 가격이 160배가 되는 이유는 무엇일까요?  
그것은 바로 **'브랜드(Brand)'**의 힘입니다.

**"그렇다면 우리도 직접 브랜드를 만들어보면 어떨까?"**

이런 상상에서 **MODIF**가 시작되었습니다.

* **Mode** (옷, 패션) + **If** (만약에)
* **Modify your mode** : 당신의 패션을 새롭게 정의해보세요.

## 📌 Key Features

### 1. Discover (발견)
* **태그 기반 큐레이션 피드 구현**
  * Funding 상태의 프로젝트를 추천순으로 호출 및 카테고리 필터링 적용
* **실시간 펀딩 달성률 시각화**
  * 목표 금액 대비 현재 모금액 비율 계산 및 프로그레스 바(Progress Bar) UI 제공
* **즉시 피팅(Instant Fitting) 연동**
  * 상품 탐색 중 해당 의류 데이터를 유지한 채 피팅 탭으로 전환되는 프로세스 구현

### 2. My Brand (제작)
* **Generative AI 기반 디자인 스튜디오 구축**
  * OpenAI API 연동을 통한 텍스트 프롬프트 및 도안 기반 의류 이미지 생성
* **의류 물성(Physics) 커스터마이징 기능 제공**
  * 신축성, 무게감, 뻣뻣함 등 물리적 속성값(1~10) 및 세부 사이즈 조정 로직 구현
* **브랜드 런칭 프로세스 시스템화**
  * 의류 등록 개수 제한 체크 및 펀딩 레코드 생성/발행 자동화

### 3. My Fitting (피팅)
* **다중 레이어링(Layering) 합성 로직 구현**
  * 상의, 하의, 아우터 등 의류 레이어 순서(Order)에 따른 이미지 합성 처리
* **듀얼 피팅 뷰(Dual View) 모드 제공**
  * 사용자 실물 사진 기반 피팅 및 3D 마네킹 뷰 지원
* **가상 옷장 및 아카이빙 기능**
  * 관심 상품 저장(My Closet) 및 피팅 결과물 앨범 저장 기능 구현

### 4. Portfolio (관리)
* **활동 데이터 시각화 대시보드 제공**
  * 내가 만든 브랜드의 참여자 수 및 펀딩 모금액 실시간 집계 표시
* **투자(Funding) 상태 추적 시스템 구축**
  * 펀딩 참여 내역 조회 및 프로젝트 상태(성공/제작중 등) 단계별 트래킹 기능 구현
* **소셜 네트워크 관리 및 통계 제공**
  * 팔로워/팔로잉 관리 및 증감 추이 그래프 시각화
